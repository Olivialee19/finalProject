# finalProject
Inputed values for x values, y values, x limit values and y limit values should all be seperated by commas. 
Brackets are not allowed  
An example input as follows
x-values input: 5,7,3
y-values input: 9,2,6
x-limit: 1,10
y-limit: 1,10
Plot title: Graph
X-axis title: Time
Y-axis title: Height
